# Gravity ZERO 教室管理アプリ

トランポリン教室 Gravity ZERO の会員が、会員IDを入力してチケット残数を確認し、「1回受講する」ボタンで1回ずつ手動消化するアプリです。

## 他アプリとの違い(重要)

このプロジェクトは開発当初「デジタル回数券」という名前で作業を始めたため、名称が似た別アプリと混同しやすいです。

- **本リポジトリ(教室管理アプリ)**: 会員IDを入力→残数確認→「1回受講する」を1回ずつ手動で押して消化するシンプルな構成。バックエンドは本リポジトリに含まれない独立したGoogle Apps Script(JSONP方式)。
- **`digital-kaisuuken` + `gravity-zero-assets`(現行デジタル回数券)**: 別のGitHubリポジトリ2つで構成される、より新しい回数券アプリ。1〜4回をまとめて選んで消化するボタンや、紙吹雪・ワープ演出などのアニメーションがある。GASはclasp管理のJSON API。

「教室管理アプリ」と言われたら本リポジトリ、「デジタル回数券」と言われた場合は文脈次第(本リポジトリのタイトルにも同じ文字列が残っているため要注意)。

## ファイル構成

- `index.html` — 実際に公開・運用されている本体(GitHub Pages配信)。他の2つのHTMLファイルは初期コミット以降更新されていない古いバックアップで、現在は使われていない。
  - `教室管理ロゴあり_全文.html`
  - `教室_アイコン修正版.html`
- `manifest.json` / `sw.js` — PWA(ホーム画面追加)用の設定
- `グラゼロ教室アイコン.jpg` — ホーム画面アイコン/favicon

## 実装済みの主な機能

- 残数が0になった瞬間(4回目消化のタイミング)に、Canvasベースの花火アニメーションを自動再生(`index.html`内の`playFireworkEffect`まわり)
